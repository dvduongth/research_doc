📜 Constitution — 7 Nguyên Tắc Cốt Lõi
PRINCIPLE 1
Workspace as Truth
ccn2_workspace/ là nguồn sự thật duy nhất. Không agent nào giữ state trong memory.
PRINCIPLE 2
Push-based
Agents chủ động push output. Không poll nhau — chỉ poll shared workspace.
PRINCIPLE 3
Idempotent
Chạy lại cùng input → cùng output. Mọi operation an toàn khi retry.
PRINCIPLE 4
Single Responsibility
Mỗi agent làm đúng 1 việc. Không agent nào kiêm nhiệm role của agent khác.
PRINCIPLE 5
State Tracking
Mọi output được track trong state JSON với hash, timestamp, status.
PRINCIPLE 6
Quality Gate
Không pass gate = không tiến sang phase tiếp theo. Rubric là law.
PRINCIPLE 7
Evidence Before Completion
Không claim done nếu không có artifact. File phải tồn tại.
