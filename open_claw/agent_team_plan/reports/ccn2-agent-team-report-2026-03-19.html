10 System Constitution — 7 Principles

Principle 1
GDD-First — Không code nếu không có GDD đã qua Review (score ≥ 70)

Agent Sovereignty — Mỗi agent chịu trách nhiệm độc lập, không phụ thuộc real-time
Principle 3
Hash-Based Idempotency — Chỉ reprocess khi file thực sự thay đổi (hash mismatch)

Log & Skip — Agent không crash cron — mọi lỗi được log, cron luôn exit 0

Human Gate — Done ≠ Flagged — cần human decision để resolve Flagged
Principle 6 NEW
48h Watchdog — GDD stuck > 48h → Telegram alert + pipeline-watch report
