Human ──→ tạo file → concepts/feature.md ↓ (cron 15 phút) agent_gd ──scan→ hash change → tạo GDD ──→ design/GDD-FEATURE-*.md ↓ (cron 15 phút) agent_dev ──scan→ hash change → analyze + dispatch ──→ .state/dispatched.json ↓ (cron 15 phút x3 agents) agent_dev_server/client/admin ──poll→ implement ──→ playtest/ ↓ (cron 15 phút) agent_qc ──poll→ review ──→ reports/ Total latency: 60–90 phút end-to-end

Ma trận rủi ro
Rủi ro Xác suất Impact Mitigation
OpenClaw CLI chưa có --agent flag Cao BLOCKING R5 Verify T5.1 trước tiên. Fallback: dùng direct bash prompt thay CLI.
Action logs quá lớn — disk space Trung bình Moderate 7-day retention policy. agent_qc auto-cleanup files cũ hơn 7 ngày.
Agent không follow chain call protocol Trung bình High Cập nhật SOUL.md + test với 1 agent (agent_gd) trước, rồi roll out.
Ticket tracker conflict (2 agents ghi cùng lúc) Thấp Low Last-write-wins strategy. Mỗi agent ghi field của riêng mình (pipeline_steps[index]).
Breaking change làm hỏng cron cũ Thấp High Giữ cron intact làm fallback. Chỉ thêm chain calls ON TOP, không xóa cron.
Dashboard HTML quá phức tạp — khó maintain Trung bình Low Dùng vanilla JS (không framework). Tab structure rõ ràng. Comment đầy đủ.
