# reports/ — Quality & Pipeline Reports

**Verita (agent_qc) viết vào đây. Human đọc ở đây.**

---

## File Types

| File | Tạo khi | Generated by | Format |
|------|---------|--------------|--------|
| `dashboard.html` | Mỗi lần agent_qc chạy | Verita | HTML (auto-refresh 15 phút) |
| `code-review-<feature>-<YYYY-MM-DD-HH-mm>.md` | GDD hoàn thành InQC | Verita | Structured review |
| `testcases-<feature>.md` | GDD mới được detect | Verita | Markdown checklist |
| `playtest-smoke-<YYYY-MM-DD-HH-mm>.md` | Sau mỗi smoke test | Verita | Pass/Fail per endpoint |
| `pipeline-watch-<YYYY-MM-DD-HH-mm>.md` | GDD stuck >48h | Verita | Alert report |
| `bug-summary-<YYYY-MM-DD>.md` | Mỗi QC cycle | Verita | Open/fixed/closed counts |

---

## Dashboard (đọc đầu tiên)

Mở `reports/dashboard.html` trong browser:
- Tự refresh mỗi 15 phút
- Hiển thị: pipeline verdict, 7 smoke checks (C1–C7), stuck GDDs, recent bugs
- **C7_playtest** = `SKIP` nếu server chưa build (bình thường — không ảnh hưởng verdict)

---

## Đọc Reports theo mục đích

| Mục đích | File cần đọc |
|---------|-------------|
| Tổng quan nhanh | `dashboard.html` |
| Review kết quả feature | `code-review-<feature>-<latest>.md` |
| Test cases để manual QA | `testcases-<feature>.md` |
| Trạng thái playtest server | `playtest-smoke-<latest>.md` |
| GDD bị stuck | `pipeline-watch-<latest>.md` |
| Tình hình bugs | `bug-summary-<latest>.md` hoặc `.state/bug-tracker.json` |

---

## Score Gates (Verita evaluation)

| Metric | Pass | Fail → action |
|--------|------|----------------|
| GDD completeness | 10 sections đủ | Flag GDD, notify |
| Code review score | ≥ 70/100 | Comment trong code-review-*.md |
| Smoke test C1–C6 | Tất cả PASS | Verdict = BROKEN nếu bất kỳ FAIL |
| Smoke test C7 | PASS/SKIP | SKIP không ảnh hưởng verdict |
| Bug verify | `fixed` → `closed` | `not_fixed` → reopen bug |

---

## Alerts

- Verita gửi Telegram alert khi:
  - Pipeline verdict = **BROKEN**
  - GDD stuck > 48h
  - Bug không được fix sau 24h
- Xem `docs/RUNBOOK-agent-errors.md` nếu alert liên tục
